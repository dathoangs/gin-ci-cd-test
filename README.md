# Gin Go Playground

A small project to experiment with **Go**, **Gin**, and **GitHub Actions** for automatic Docker builds.

## Overview

- **Language:** Go  
- **Framework:** Gin  
- **Purpose:** Learn Go, build a simple web app, and play with GitHub Actions for automation.

## Getting Started

### Prerequisites

- Go installed (v1.25.4+ recommended)  
- Git  
- Docker (optional, for building the image locally)

### Installation

```bash
git clone git@github.com:dathoangs/gin-ci-cd-test.git
cd gin-ci-cd-test
go mod tidy
```

### Running the App
```bash
go run main.go
```

Visit http://localhost:8080 to see it in action.

### Experimentation

- Testing GitHub Actions workflows
- Auto-building Docker images
- Learning deployment basics
